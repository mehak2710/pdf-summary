# 📄 PDF Summarizer Chatbot (15-Line Summary)

A simple Streamlit app that summarizes a PDF (up to 500 lines) into **exactly 15 lines** using **Groq’s LLaMA3** model.

## 🚀 Features

- Upload a PDF 📤  
- Extract text 🔍  
- Summarize to 15 lines 🧠  
- Powered by Groq + Streamlit ⚡
  
## 🛠 Tech Stack

- Python + Streamlit
- Groq (LLaMA3-8B)
- PyPDF2
- python-dotenv
